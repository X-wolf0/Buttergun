# Buttergun 题库管理系统
> 为编程学习者打造的一站式 OJ 题库工具，从题目爬取到刷题练习全流程覆盖

---

## 📌 项目简介
Buttergun 是一个长期迭代的开源项目，旨在构建一个集 **题目爬取、题库管理、刷题练习、AI辅助** 于一体的本地 OJ 学习平台。
当前阶段已完成核心的题目爬虫模块，后续将逐步扩展数据库存储、刷题、批改等功能。

---

## ✨ 项目定位
- **面向用户**：编程学习者、算法竞赛爱好者
- **核心目标**：打破 OJ 平台限制，构建可离线、可自定义的本地题库
- **技术栈**：Python + SQLite + Requests + BeautifulSoup4（后续扩展本地AI模块）

---

## 🚀 已实现功能
### 1. OJ 题目爬虫（当前版本核心）
- ✅ **分页遍历**：自动遍历青海大学 OJ 题目列表，批量获取题目ID
- ✅ **精准内容提取**：解决网页 `<span>` 碎片化问题，完整爬取题目描述、输入、输出、样例输入/输出
- ✅ **格式修复**：还原样例代码块的换行与排版，输出结构化题目内容
- ✅ **防反爬机制**：合理设置请求间隔，避免被服务器拦截

---

## 📋 后续开发规划
- [ ] **本地题库模块**：将爬取的题目存入 SQLite 数据库，支持按ID、难度、标签查询筛选
- [ ] **刷题练习系统**：实现本地代码运行、样例测试与自动批改功能
- [ ] **错题本与进度管理**：记录做题状态，生成个人刷题统计
- [ ] **AI辅助模块**：接入本地大模型，提供解题思路、知识点解析与代码优化建议
- [ ] **可视化界面**：开发简易GUI，支持题库浏览、刷题、进度查看
